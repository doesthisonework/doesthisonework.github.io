I can't imagine anything more awesome than making things for the internet.

Luckily, that's what I do.

Front end, back end, I love the full stack from top to bottom. I like server-side code the same way I like my front end UI: clean, elegant and user friendly, and I'm ready to get my hands dirty working with both.

Some of the languages and technologies I work with are: Ruby on Rails, JavaScript, SQL, HTML5, CSS3, CoffeeScript, Sass, RSpec, Capybara, Backbone.js, Node.js, jQuery, Bootstrap, and Git. And I'm always looking to add new things to my bag of tricks. Next up: Angular.js.

I'm all about code, but not only about code. Among other things, I've also been a writer, editor, wholesale sales manager, e-commerce expert, and wine educator.
